# amend

Re-edit the last commit message using the same guided prompts.

**Alias:** `devflow a`

## Description

Pre-fills all fields from the existing commit message, letting you modify any part of the conventional commit (type, scope, ticket, subject, body). Also includes any currently staged changes in the amend.

## Options

| Option | Description |
|--------|-------------|
| `--dry-run` | Preview the amended message without executing |
